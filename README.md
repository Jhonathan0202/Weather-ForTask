# Weather ForTask
Um site Single Page Application (SPA) que tem como objetivo criar tarefas, separadas por blocos de tarefas, e notas.

## Constuído com
- [Claude AI](https://claude.ai/) para melhorar a UI e auxiliar o desenvolvimento.
- Copilot AI do Visual Studio Code para revisão de código e algumas sugestões.
- [Google Fonts](https://fonts.google.com/) para fontes e icones.
